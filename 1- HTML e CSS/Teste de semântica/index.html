O que é Placa Mãe?:

É a placa de circuito central que conecta e permite a comunicação entre todos os outros componentes do computador.

O que é o Processador?:

Considerado o "cérebro" do computador, é responsável por executar comandos, processar dados e realizar cálculos, influenciando diretamente a velocidade das tarefas.

O que é a Memória Ram?:

Armazena temporariamente os dados dos programas em execução. Essa memória é volátil, ou seja, os dados são apagados quando o computador é desligado.

O que é Disco Rígido ou SSD?:

Local de armazenamento permanente onde todos os arquivos e dados do sistema são guardados a longo prazo, mesmo com o computador desligado.

O que é a Placa de Vídeo?:

Processa e exibe as imagens e vídeos no monitor, melhorando o desempenho visual do computador.

O que é a Fonte de Alimentação?:

Fornece a energia elétrica necessária para que todos os componentes internos do computador funcionem.
